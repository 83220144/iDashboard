# iDashboard - Laravel5.3基本的权限管理系统

Vue2.0版本只完成了部分，用户管理，权限管理以及角色管理，其他的没有时间做了。重复的增删改查不想写，代码共参考，近期会出Laravel5.3和Vue2.0的视频教程。请关注博客！

## 安装
迁移文件及数据填充：

```php
php  artisan migrate --seed
```

在项目根目录下执行：

```
npm install

# 开发环境
npm run dev
```
